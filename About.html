tomcat.moahub.org:

The website is powered by Tomcat and Guacamole. Tomcat is an open-source web server and servlet container. Guacamole is an open-source remote desktop gateway. Tomcat and Guacamole are licensed under the Apache License 2.0. For more information about Tomcat and Guacamole, and their licensing, please visit The offical Apache Website.

proxmox.moahub.org:

Proxmox is an open-source virtualization platform. Proxmox is licensed under the GNU Affero General Public License. For more information about Proxmox and its licensing, please visit the official Proxmox website.

phoneinfoga.moahub.org

PhoneInfoga is an open-source tool designed to gather information about phone numbers. Using PhoneInfoga to collect information about someone without their permission might be against the law in some places. It's really important to understand and follow the rules before using this tool. PhoneInfoga is provided under the GNU General Public License (GPL) v3.0, which means it's free to use, change, and share, but you also have to share any changes you make to it. You can find the official PhoneInfoga website and instructions at https://sundowndev.github.io/phoneinfoga/.
